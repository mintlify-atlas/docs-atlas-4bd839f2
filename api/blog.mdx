# Blog API

The Blog API endpoint allows you to retrieve published blog posts including their authors, post types, categories, and media gallery.

## Endpoint

### GET /api/blog

Retrieve a paginated list of published blog posts.

**Authentication:** Not required (public endpoint)

**Route Name:** `api.blog`

## Request

### Query Parameters

<ParamField query="page" type="integer" default="1">
  The page number to retrieve
</ParamField>

<ParamField query="per_page" type="integer" default="15">
  Number of posts per page (Laravel default pagination)
</ParamField>

### Example Request

```bash
curl "https://your-domain.com/api/blog?page=1" \
  -H "Accept: application/json"
```

```bash
# With pagination
curl "https://your-domain.com/api/blog?page=2&per_page=10" \
  -H "Accept: application/json"
```

## Implementation Details

### Controller

Implemented in `app/Http/Controllers/API/ApiBlogController.php`:

```php
class ApiBlogController extends Controller
{
    public function __invoke(Request $request)
    {
        $posts = Post::where('is_published', true)
            ->with(['author', 'type', 'categories'])
            ->orderBy('is_featured', 'DESC')
            ->paginate();

        return new PostCollection($posts);
    }
}
```

### Filtering Logic

- **Published Only**: Only posts with `is_published = true` are returned
- **Eager Loading**: Related data (author, type, categories) is loaded to prevent N+1 queries
- **Ordering**: Posts are ordered by featured status (featured posts first)
- **Pagination**: Results are paginated using Laravel's default pagination (15 per page)

### Media Collections

Posts use Spatie Media Library with the following configuration:

**Collection:** `gallery` (stored on S3)

**Conversions:**
- `main` - 1200x620px, quality 85%, cropped, with responsive images
- `hero` - 1920x1080px, quality 85%, cropped

Both conversions include sharpening (10) for better image quality.

### Resource Transformation

The `PostResource` class (`app/Http/Resources/PostResource.php`) transforms each post to include processed media URLs:

```php
class PostResource extends JsonResource
{
    public function toArray(Request $request): array
    {
        $data = parent::toArray($request);

        $data['media'] = $this->getMedia('gallery')->map(
            fn($m) =>
            new MediaResource($m, 'main', [
                'main' => ['width' => 1200, 'height' => 620],
                'hero' => ['width' => 1920, 'height' => 1080]
            ], ['hero'])
        );

        return $data;
    }
}
```

## Usage Examples

### JavaScript (Fetch API)

```javascript
fetch('https://your-domain.com/api/blog')
  .then(response => response.json())
  .then(data => {
    console.log(`Total posts: ${data.meta.total}`);
    data.data.forEach(post => {
      console.log(`${post.name} by ${post.author.name}`);
    });
  })
  .catch(error => console.error('Error:', error));
```

### JavaScript (Axios)

```javascript
import axios from 'axios';

const response = await axios.get('https://your-domain.com/api/blog', {
  params: {
    page: 1,
    per_page: 10
  }
});

const posts = response.data.data;
const pagination = response.data.meta;

console.log(`Showing ${pagination.from}-${pagination.to} of ${pagination.total} posts`);
```

### PHP

```php
$response = Http::get('https://your-domain.com/api/blog', [
    'page' => 1,
    'per_page' => 10
]);

$posts = $response->json()['data'];

foreach ($posts as $post) {
    echo $post['name'] . " by " . $post['author']['name'] . "\n";
}
```

### Python

```python
import requests

response = requests.get('https://your-domain.com/api/blog', params={
    'page': 1,
    'per_page': 10
})

data = response.json()

for post in data['data']:
    print(f"{post['name']} by {post['author']['name']}")

print(f"Total posts: {data['meta']['total']}")
```
