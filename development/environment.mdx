## Overview

EduStack Smart uses environment variables to configure the application. All configuration is stored in the `.env` file in the project root.

## Initial Setup

Copy the example environment file:

```bash
cp .env.example .env
```

Generate an application key:

```bash
php artisan key:generate
```

## Application Settings

### APP_NAME

**Default:** `Laravel`
**Type:** String

The name of your application. Used in emails, notifications, and UI.

```env
APP_NAME="EduStack Smart"
```

### APP_ENV

**Default:** `local`
**Type:** String
**Options:** `local`, `production`, `staging`, `testing`

The application environment. Affects error reporting, debugging, and caching behavior.

```env
APP_ENV=local
```

### APP_KEY

**Default:** (empty)
**Type:** String (base64 encoded)

The encryption key for the application. Generated automatically by `php artisan key:generate`.

<Warning>
**Never commit your APP_KEY to version control!** This key is used to encrypt session data, cookies, and other sensitive information.
</Warning>

```env
APP_KEY=base64:...
```

### APP_DEBUG

**Default:** `true`
**Type:** Boolean

Enables detailed error messages and stack traces.

<Warning>
**Always set to `false` in production!** Detailed error messages can expose sensitive information.
</Warning>

```env
APP_DEBUG=true  # Development
APP_DEBUG=false # Production
```

### APP_URL

**Default:** `http://localhost`
**Type:** URL

The base URL for your application. Used for generating URLs and asset paths.

```env
APP_URL=http://localhost:8000
# Production:
APP_URL=https://yourdomain.com
```

## Maintenance Mode

### APP_MAINTENANCE_DRIVER

**Default:** `file`
**Type:** String
**Options:** `file`, `cache`, `database`

The storage driver for maintenance mode state.

```env
APP_MAINTENANCE_DRIVER=file
```

### APP_MAINTENANCE_STORE

**Default:** (commented out)
**Type:** String

The cache store to use when `APP_MAINTENANCE_DRIVER` is set to `cache` or `database`.

```env
# APP_MAINTENANCE_STORE=database
```

## Performance

### PHP_CLI_SERVER_WORKERS

**Default:** `4`
**Type:** Integer

The number of worker processes for the PHP built-in server.

```env
PHP_CLI_SERVER_WORKERS=4
```

### BCRYPT_ROUNDS

**Default:** `12`
**Type:** Integer

The cost factor for bcrypt password hashing. Higher values increase security but slow down hashing.

```env
BCRYPT_ROUNDS=12 # Development/Production
BCRYPT_ROUNDS=4  # Testing (faster)
```

## Database Configuration

### DB_CONNECTION

**Default:** `sqlite`
**Type:** String
**Options:** `sqlite`, `mysql`, `pgsql`, `sqlsrv`

The database connection to use.

```env
DB_CONNECTION=sqlite
```

### DB_HOST

**Default:** `127.0.0.1` (commented out)
**Type:** String

The database host (for MySQL/PostgreSQL).

```env
DB_HOST=127.0.0.1
```

### DB_PORT

**Default:** `3306` (commented out)
**Type:** Integer

The database port.

```env
DB_PORT=3306      # MySQL
DB_PORT=5432      # PostgreSQL
```

### DB_DATABASE

**Default:** `laravel` (commented out)
**Type:** String

The database name. For SQLite, this is the path to the database file.

```env
# SQLite (default)
# Database file: database/database.sqlite

# MySQL/PostgreSQL
DB_DATABASE=edustack_smart
```

### DB_USERNAME

**Default:** `root` (commented out)
**Type:** String

The database username.

```env
DB_USERNAME=root
```

### DB_PASSWORD

**Default:** (empty, commented out)
**Type:** String

The database password.

```env
DB_PASSWORD=your_password
```

## Session Configuration

### SESSION_DRIVER

**Default:** `database`
**Type:** String
**Options:** `file`, `cookie`, `database`, `apc`, `memcached`, `redis`, `array`

The session storage driver.

```env
SESSION_DRIVER=database
```

### SESSION_LIFETIME

**Default:** `120`
**Type:** Integer (minutes)

The session lifetime in minutes.

```env
SESSION_LIFETIME=120
```

### SESSION_ENCRYPT

**Default:** `false`
**Type:** Boolean

Whether to encrypt session data.

```env
SESSION_ENCRYPT=false
```

### SESSION_PATH

**Default:** `/`
**Type:** String

The session cookie path.

```env

```

### SESSION_DOMAIN

**Default:** `null`
**Type:** String

The session cookie domain.

```env

# Or for production:
```

## Broadcasting, Cache, and Queue

**Default:** `log`
**Type:** String
**Options:** `log`, `redis`, `pusher`, `reverb`, `null`

### FILESYSTEM_DISK

**Default:** `local`
**Type:** String
**Options:** `local`, `public`, `s3`, `s3-public`

The default filesystem disk.

```env

```

### QUEUE_CONNECTION

**Default:** `database`
**Type:** String
**Options:** `sync`, `database`, `redis`, `sqs`, `beanstalkd`, `null`

The queue connection for background jobs.

```env
QUEUE_CONNECTION=database        # Development/Production
QUEUE_CONNECTION=sync            # Testing (immediate execution)
```

### CACHE_STORE

**Default:** `database`
**Type:** String
**Options:** `apc`, `array`, `database`, `file`, `memcached`, `redis`, `dynamodb`, `octane`

The cache storage driver.

```env
CACHE_STORE=database
```

### CACHE_PREFIX

**Default:** (commented out)
**Type:** String

Prefix for cache keys.

```env
# CACHE_PREFIX=edustack
```

### MAIL_PORT

**Default:** `2525`
**Type:** Integer

```env

MAIL_PORT=465        # SSL
```

### MAIL_USERNAME

**Default:** `null`
**Type:** String

The SMTP username.

```env
MAIL_USERNAME=your_username
```

### MAIL_PASSWORD

**Default:** `null`
**Type:** String

The SMTP password.

```env

```

**Default:** (empty)
**Type:** String

The disk for media library files (Spatie Media Library).

## Configuration Caching

In production, cache your configuration for better performance:

```bash
# Cache configuration
php artisan config:cache

# Clear configuration cache
php artisan config:clear
```

<Warning>
**When configuration is cached, the `.env` file is not loaded!** You must clear the cache after changing environment variables.
</Warning>

## Environment-Specific Configuration

### Development (.env.local)

```env
APP_ENV=local
APP_DEBUG=true
APP_URL=http://localhost:8000

DB_CONNECTION=sqlite

MAIL_MAILER=log
QUEUE_CONNECTION=database
BROADCAST_CONNECTION=log

LOG_LEVEL=debug
```

### Production (.env.production)

```env
APP_ENV=production
APP_DEBUG=false
APP_URL=https://yourdomain.com

DB_DATABASE=your_db_name
DB_USERNAME=your_db_user
DB_PASSWORD=your_db_password

MAIL_MAILER=smtp
MAIL_HOST=smtp.yourmailprovider.com
MAIL_PORT=587
MAIL_USERNAME=your_mail_username

MAIL_FROM_ADDRESS="noreply@yourdomain.com"

QUEUE_CONNECTION=redis
BROADCAST_CONNECTION=reverb
CACHE_STORE=redis

LOG_LEVEL=warning
```

### Testing (.env.testing)

```env

CACHE_STORE=array

MAIL_MAILER=array

```

## Security Best Practices

1. **Never commit `.env` to version control** - Add it to `.gitignore`
2. **Use strong, unique APP_KEY** - Generate with `php artisan key:generate`
3. **Disable debug in production** - Set `APP_DEBUG=false`
4. **Use HTTPS in production** - Set `APP_URL=https://...`
5. **Secure database credentials** - Use strong passwords
6. **Restrict file permissions** - `chmod 600 .env`
7. **Use environment variables for secrets** - Never hardcode credentials
