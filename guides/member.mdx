# Member Guide

Members in EduStack Smart have special privileges to create and manage blog posts and events. This guide covers all member-specific features for content creation and event management.

## Overview

As a member, you can:
- Create and publish blog posts
- Manage blog post categories
- Create and manage events
- Upload images and media
- Edit content with a visual editor
- Control content visibility (publish/hide)

## Creating Blog Posts

### Starting a New Blog Post

<Steps>
  <Step title="Navigate to Blog">
    Click on "Blog" in the main navigation menu.
  </Step>
  
  <Step title="Create New Post">
    Click the "Crear Post" (Create Post) button.
  </Step>
  
  <Step title="Fill Post Details">
    Complete the blog post creation form with all required information.
  </Step>
  
  <Step title="Select Categories">
    Choose at least one category that describes your post content.
  </Step>
  
  <Step title="Upload Images">
    Add at least one image for your post using the drag-and-drop zone.
  </Step>
  
  <Step title="Submit">
    Click "Crear Post" (Create Post) to save your blog post.
  </Step>
</Steps>

### Blog Post Information Fields

#### Basic Information

- **Nombre** (Name): Post title
  - Should be clear and descriptive
  - Minimum 3 characters
  - This appears as the main heading

- **Resumen** (Summary): Post summary or excerpt
  - Brief description of the post content
  - Minimum 100 characters
  - Appears in post listings and previews

- **Tipo de Post** (Post Type): Content category
  - Select from predefined types
  - Examples: Article, Tutorial, News, Announcement
  - Default: Type 1

#### Categorization

- **Categorías** (Categories): Content topics
  - Check at least one category
  - Multiple categories can be selected
  - Categories help users find related content
  - Examples might include: Technology, Science, Arts, Sports

#### Visual Content

- **Imágenes** (Images): Post images
  - Upload one or more images
  - Drag and drop into the upload zone
  - At least one image is required
  - First image is used as the featured image

<Note>
  All fields marked as required must be completed before you can create the post.
</Note>

## Managing Blog Posts

### Editing an Existing Post

<Steps>
  <Step title="Find Your Post">
    Navigate to "Blog" and locate the post you want to edit.
  </Step>
  
  <Step title="Open Edit Page">
    Click on the post and then click "Editar" (Edit).
  </Step>
  
  <Step title="Update Information">
    The edit screen is divided into:
    - **Left Panel**: Form with post details (name, summary, type)
    - **Right Panel**: Categories management and publishing options
  </Step>
  
  <Step title="Save Changes">
    Click the update button to save your modifications.
  </Step>
</Steps>

### Managing Post Categories

On the edit page, the right sidebar contains a category management section:

<Steps>
  <Step title="View Categories">
    The "Categorías" section shows all available categories with checkboxes.
  </Step>
  
  <Step title="Select/Deselect">
    - Check boxes to add categories
    - Uncheck boxes to remove categories
    - At least one category must remain selected
  </Step>
  
  <Step title="Auto-Save">
    Category changes are saved automatically when you update the post.
  </Step>
</Steps>

## Publishing and Visibility

### Understanding Post Status

Blog posts have two visibility states:

- **Publicado** (Published): Visible to all users in the blog section
- **Oculto** (Hidden): Only visible to you, not shown in public listings

### Publishing or Hiding a Post

<Steps>
  <Step title="Go to Edit Page">
    Navigate to your post and click "Editar" (Edit).
  </Step>
  
  <Step title="Find Status Card">
    In the right sidebar, locate the "Estado del post" (Post Status) card.
  </Step>
  
  <Step title="Check Current Status">
    The card shows:
    - Current status in colored text (green for Published, red for Hidden)
    - Explanation of each status
  </Step>
  
  <Step title="Toggle Status">
    Click the appropriate button:
    - "Publicar Post" (Publish Post) - makes the post visible to everyone
    - "Ocultar Post" (Hide Post) - removes from public view
  </Step>
</Steps>

<Tip>
  Keep posts hidden while drafting. Publish only when content is complete and reviewed.
</Tip>

## Editing Post Content

### Using the Visual Content Editor

<Steps>
  <Step title="Access Content Editor">
    From the post edit page, click "Editar" in the "Editar Contenido" card.
  </Step>
  
  <Step title="Build Your Content">
    The visual editor provides:
    - Rich text formatting (bold, italic, headings)
    - Image insertion and positioning
    - Code blocks for technical content
    - Custom layouts and sections
  </Step>
  
  <Step title="Drag and Drop">
    Rearrange content blocks by dragging them to new positions.
  </Step>
  
  <Step title="Save">
    Save your changes. Updated content appears immediately on the published post.
  </Step>
</Steps>

## Managing Post Gallery

The gallery section appears at the bottom of the edit page:

### Adding More Images

1. Scroll to the "Galería" section
2. Click the upload area or drag and drop new images
3. Multiple images can be added
4. Images upload and save automatically

### Removing Images

1. Hover over an image in the gallery
2. Click the delete button
3. Confirm removal (note: at least one image must remain)

## Creating Events

### Starting a New Event

<Steps>
  <Step title="Navigate to Events">
    Click on "Eventos" (Events) in the main navigation.
  </Step>
  
  <Step title="Create New Event">
    Click "Crear Evento" (Create Event).
  </Step>
  
  <Step title="Complete Event Form">
    Fill in all event details including dates, location, and pricing.
  </Step>
  
  <Step title="Upload Logo">
    Add an event logo or image.
  </Step>
  
  <Step title="Submit">
    Click "Crear Evento" (Create Event) to save.
  </Step>
</Steps>

### Event Information Fields

#### Basic Details

- **Nombre** (Name): Event title
- **Resumen** (Summary): Event description
- **Logo**: Event image/logo (required, single image)

#### Location and Timing

- **Ubicación** (Location): Physical or virtual location
- **Coordinates**: Map location (latitude/longitude)
  - Click on a map to set the location
  - Marker shows exact event location

- **Fecha de Inicio** (Start Date): When the event begins
- **Fecha de Fin** (End Date): When the event ends

#### Registration

- **Inicio de Registro** (Registration Start): When registration opens
- **Fin de Registro** (Registration End): Registration deadline

#### Pricing

- **Es Gratis** (Is Free): Check if the event is free
  - If checked, price fields are disabled
  - If unchecked, enter pricing information

- **Precio** (Price): Event cost (if not free)
- **Descuento** (Discount): Percentage off (0-100%)

<Note>
  Ensure registration dates fall before the event start date, and end registration before the event begins.
</Note>

## Managing Events

### Editing an Event

<Steps>
  <Step title="Find Your Event">

  </Step>
  
  <Step title="Open Edit Mode">
    Click on the event and then "Editar" (Edit).
  </Step>
  
  <Step title="Update Details">
    Modify any event information:
    - Basic details (name, summary, logo)
    - Dates and times
    - Location
    - Pricing
  </Step>
  
  <Step title="Save Changes">
    Click the update button to save modifications.
  </Step>
</Steps>

### Publishing Events

Similar to blog posts, events can be published or hidden:

<Steps>
  <Step title="Access Event Options">
    Go to the event edit page.
  </Step>
  
  <Step title="Find Status Card">
    Locate the "Estado del evento" (Event Status) card in the sidebar.
  </Step>

  </Step>
</Steps>

### Editing Event Content

Events also support rich content editing:

1. Click "Editar Contenido" in the event options

   - Detailed event information
   - Schedules and agendas
   - Speaker information
   - Special instructions
3. Save changes to update the public event page

## Troubleshooting

### Cannot Create Post

**Common issues**:
- No category selected
- Summary too short (less than 100 characters)
- No image uploaded

**Solution**: Check for validation error messages below each field and correct them.

### Event Dates Not Saving

**Solution**: Ensure:
- Registration start is before registration end
- Registration end is before event start
- Event start is before event end
- All dates are in the future

### Image Upload Fails

**Solutions**:
- Check file size (reduce if too large)
- Verify file format (use JPG or PNG)
- Try uploading a single image first
- Check your internet connection

### Content Not Appearing After Publishing

**Solution**: 
1. Verify status shows "Publicado" (Published)
2. Clear your browser cache
3. Try viewing in an incognito/private window
4. Check that all required fields are completed
